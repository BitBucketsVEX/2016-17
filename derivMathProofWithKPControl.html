`v = K_p * epsilon(t)`

If error = `epsilon(t + 1)`, and the distance desired = `n`

`n - K_p * epsilon(t) * \Deltat = epsilon(t + 1)`

To get an error of 0, move `epsilon(t + 1)` to the other side of the equation

`n - K_p * epsilon(t) * \Deltat - epsilon(t + 1) = 0`

`n - K_p * epsilon(t) * \Deltat - (epsilon(t + 1) * \Deltat) / (\Deltat) = 0`

`n - \Deltat * (K_p * epsilon(t) - (epsilon(t + 1)) / (\Deltat)) = 0`

New velocity: `K_p * epsilon(t) - (epsilon(t + 1)) / (\Deltat)`

`v_n = K_p * epsilon(t) - (epsilon(t + 1)) / (\Deltat)`

`v_n \approx epsilon(t) * (K_p - 2 / (\Deltat)) + (epsilon(t - 1)) / (\Deltat)`
